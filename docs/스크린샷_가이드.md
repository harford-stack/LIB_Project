# 스크린샷 가이드

포트폴리오용 스크린샷을 촬영할 때 참고하세요.

---

## 📸 필수 캡처 페이지

### 1. 지도 페이지 (메인) ⭐⭐⭐
**우선순위**: 최우선  
**캡처 포인트**:
- 지도와 마커들이 표시된 전체 화면
- 위젯들 (시계, 날씨, 나침반, 레이더)이 모두 보이는 화면
- 마커 상세 패널이 열린 화면
- 카테고리 필터가 보이는 화면

**추천 구성**:
- 지도 중앙에 마커들이 표시된 상태
- 왼쪽 상단에 위젯 4개 모두 보이기
- 오른쪽에 마커 상세 패널 열기
- 여러 카테고리의 마커가 보이도록 설정

---

### 2. 로그인/회원가입 페이지 ⭐⭐
**우선순위**: 높음  
**캡처 포인트**:
- 레트로 테마가 잘 보이는 화면
- 별 배경 애니메이션이 보이는 화면
- 네온 색상이 강조된 화면

**추천 구성**:
- 로그인 페이지: 입력 필드와 버튼이 보이는 화면
- 회원가입 페이지: 아이디 중복 체크 버튼이 보이는 화면

---

### 3. 피드 페이지 ⭐⭐⭐
**우선순위**: 최우선  
**캡처 포인트**:
- 마커 카드들이 표시된 화면
- 탭 전환 (팔로잉/탐색) 화면
- 정렬 옵션 (NEW/HOT) 화면
- 무한 스크롤이 작동하는 화면

**추천 구성**:
- 탐색 탭 선택 상태
- HOT 정렬 선택 상태
- 마커 카드 3-4개가 보이도록 스크롤
- 카드에 이미지, 좋아요, 댓글 수가 보이도록

---

### 4. 프로필 페이지 ⭐⭐
**우선순위**: 높음  
**캡처 포인트**:
- 프로필 정보 (닉네임, 상태 메시지, 프로필 이미지)
- 통계 (마커 수, 팔로워 수, 팔로잉 수)
- 마커 그리드 (4열 레이아웃)
- 팔로워/팔로잉 목록 모달

**추천 구성**:
- 내 프로필 페이지: 프로필 수정 버튼이 보이는 화면
- 다른 사용자 프로필: 팔로우 버튼과 채팅 버튼이 보이는 화면
- 마커 그리드가 4열로 정렬된 화면
- 팔로워 목록 모달이 열린 화면

---

### 5. 채팅 페이지 ⭐⭐⭐
**우선순위**: 최우선  
**캡처 포인트**:
- 채팅방 목록과 메시지가 보이는 화면
- 실시간 메시지 전송 화면
- 메시지 정렬 (내 메시지 오른쪽, 상대방 메시지 왼쪽)
- 채팅 알림 배지

**추천 구성**:
- 채팅방 목록이 왼쪽에 보이는 화면
- 메시지가 오고 가는 화면
- 내 메시지는 오른쪽 (보라색), 상대방 메시지는 왼쪽 (파란색)
- 채팅방 나가기 버튼이 보이는 화면

---

## 📸 선택 캡처 페이지

### 6. 북마크 페이지 ⭐
**우선순위**: 보통  
**캡처 포인트**:
- 북마크한 마커 그리드
- 4열 레이아웃

**추천 구성**:
- 북마크한 마커 4개 이상이 보이는 화면

---

### 7. 검색 결과 페이지 ⭐
**우선순위**: 보통  
**캡처 포인트**:
- 검색 결과가 표시된 화면
- 마커 검색 결과
- 사용자 검색 결과

**추천 구성**:
- 검색창에 검색어가 입력된 상태
- 검색 결과 목록이 보이는 화면

---

### 8. 알림 목록 ⭐
**우선순위**: 보통  
**캡처 포인트**:
- 알림 배지가 표시된 헤더
- 알림 목록이 열린 화면
- 다양한 알림 타입 (좋아요, 댓글, 팔로우, 채팅)

**추천 구성**:
- 헤더의 알림 아이콘에 배지가 보이는 화면
- 알림 목록이 열려서 여러 알림이 보이는 화면

---

## 🎨 캡처 팁

### 1. 화면 구성
- **해상도**: 최소 1920x1080 권장
- **브라우저**: Chrome 또는 Edge 사용
- **전체 화면**: F11로 전체 화면 모드

### 2. 데이터 준비
- **테스트 계정**: 여러 개의 테스트 계정 생성
- **테스트 데이터**: 마커, 댓글, 좋아요 등 충분한 데이터 준비
- **채팅 데이터**: 채팅 메시지가 있는 상태로 캡처

### 3. 시각적 효과
- **애니메이션**: 별 배경 애니메이션이 보이는 상태
- **네온 효과**: 네온 색상이 잘 보이는 화면
- **위젯**: 위젯들이 모두 보이는 화면

### 4. 캡처 도구
- **Windows**: Win + Shift + S (스니핑 도구)
- **Mac**: Cmd + Shift + 4
- **브라우저 확장**: Full Page Screen Capture (전체 페이지 캡처)
